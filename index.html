Font

A tag em HTML desempenha um papel importante na página web para criar uma página web atraente e legível. A tag de fonte é usada para alterar a cor, o tamanho e o estilo de um texto. A tag de fonte base é usada para definir todo o texto para o mesmo tamanho, cor e face.

Observação: A tag font não é mais suportada no HTML5
